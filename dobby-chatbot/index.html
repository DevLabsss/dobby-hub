Commands

Prefix: /
Command Args Description
/roles — Show available contributor roles + quick claim buttons.
/claim role Claim Helper/Educator/Artist/Builder.
/faq [topic] Get short answers about rules, levels, channels.
/ping — Latency check.
/help — List all commands.

Quick Setup

  1. Create a Discord Application → Bot → copy Token.
  2. Invite with the button above (scopes: bot, applications.commands).
  3. Clone repo & set env: 
    DISCORD_TOKEN=xxxxx
GUILD_ID=your_guild
CLIENT_ID=YOUR_CLIENT_ID
ROLE_HELPER=123
ROLE_EDUCATOR=456
ROLE_ARTIST=789
ROLE_BUILDER=101112
  4. Run locally: 
    pnpm i   # or npm i
pnpm dev # or node index.js
